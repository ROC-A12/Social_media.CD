# Social Media Platform

A simple social media platform built with PHP and MySQL.

## Setup Instructions

1. Start XAMPP (Apache and MySQL)
2. Place the project in `C:\xampp\htdocs\GitHub\Social_media.CD\`
3. Run `setup.php` in your browser: `http://localhost/GitHub/Social_media.CD/setup.php`
4. Run `create_user.php` to add a test user: `http://localhost/GitHub/Social_media.CD/create_user.php`
5. Login at `http://localhost/GitHub/Social_media.CD/login.php` with:
   - Email: test@example.com
   - Password: password

## Features

- User registration and login
- Post creation with text and images
- Like and comment on posts
- Follow other users
- Admin dashboard
- Search users

## Database Tables

- users
- posts
- likes
- comments
- followers
